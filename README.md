# Task L – Final Rental Website (PrimeStay)

This folder contains the final, polished three‑page website for **PrimeStay Apartments**, combining the landing page, catalog page, and Application page into one coherent site.

---

## Pages

- `index.html` – Landing page
- `catalog.html` – Catalog with filters and modal
  popups
- `application.html` – Apply form with validation

---

### Navigation & Layout

- Unified **header, navigation bar, and footer** across all three pages
- Ensured all nav links work correctly with **relative paths**
- Added consistent spacing, typography, and button styles for a professional look

### Visual Identity

- Shared **color palette** and consistent typography across pages
- Standardized card layouts, shadows, and margins
- Improved contrast on buttons for accessibility

### Responsiveness

- Refined catalog grid to adapt smoothly on **mobile, tablet, and desktop**
- Fixed layout overflow issues and ensured modals scale properly on smaller screens

### Accessibility & Semantics

- Corrected **heading hierarchy** (`h1`, `h2`, etc.) for clarity
- Added **alt text** for all images
- Used semantic HTML elements (`header`, `nav`, `main`, `footer`) consistently
- Improved form labels and required field indicators on the order page

### Validation & Compliance

- Ran all pages through **W3C HTML Validator** and fixed errors/warnings
- Ensured consistent file references and relative paths
- Included the required `example.jpg` in the catalog page

### Order changed to application Form Improvements

- Clearer validation messages
- Properly associated labels with inputs
- Clean, accessible layout for form fields
